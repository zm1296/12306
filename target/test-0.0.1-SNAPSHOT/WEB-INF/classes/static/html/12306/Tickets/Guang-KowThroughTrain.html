注意事项
始发改签

旅客如不能按票面指定的日期和车次乘车时，在列车有能力的条件下，可办理一次提前或改晚乘车签证手续。旅客在票面乘车站办理改签时，应于票面指定乘车日期开车前30分钟办理,异地票在原购票地办理时，应于票面指定乘车8期开车前6小时办理。已办理改签的车票不再办理退票。末班车不办理改签。

退票

旅客要求退票,在票面乘车站力理时，应于票面指定乘车日期开车前2小时办理.异地票在原购票地办理时,应于票面指定乘车日期开车前6小时办理。提前2小时-2天办理退票时，按票面票价退款50%;提前3- 14天办理退票时，按票面票价退款70%;提前15天以上办理退票时，按票面票价退款100%，另核收手续费15港元(根据每月的汇率折算人民币)。

变更席别

旅客要求变更高于原票等级的列车或座席时，要核收变更区间的票价差额。变更低于原票等级的列车或席位时，车票要价差额不予退还

儿童票

年满5周岁至未满10周岁的小孩乘车时，应随同大人购买座别相同的小孩票，年满10周岁的小孩应买成人票。每一成人可免费携带一名不满5周岁 并且不单独占用座席的小孩-名，超过一名时，超过人数应买小孩票。

乘车须知

携带品大人免费携带重量为20千克(不包括残疾人随身使用的折叠式残疾人用车)，小孩(包括免费小孩)为10千克，外交人员为35千克。每件携带品的外部尺寸长宽高相加之和，不得超过160厘米;杆状物品不超过200厘米。超过规定时，请到行李包裹托运处办理托运手续。

旅客携带供玩赏的小动物，(猫 狗、猴鸟)必须装在相应的容器内，需符进出口检疫的有关规定，并保证其他旅客的安全和车内清洁卫生。应向车站办理托运(自理)手续，猫、狗猴每只核收运费港币60元，小鸟每只核收港币10元，列车上发现未起票的加倍补收运费。

禁止带入站、车内物品:危险品(包括易燃品、自燃品、爆炸品、放射性物质、腐蚀性和毒害性等有害的物品);武器和弹药;可能传播疾病或发臭的物品;可能损坏、污染车厢或其他旅客携带品的物品;海关禁止携带的物品。

口岸规定:广九直通车(广州东-九龙)口岸限定区域，旅客需在开车前45分钟到站，办理有关出境手续，进站闸口在开车前10分钟关闭，严禁无有效证件及送客人员进入。

粤港直通车车票的有效期为当日当次使用一次有效。
