中铁银通卡

  简介

1、中铁银通卡是由中铁银通支付有限公司发行的预付卡。该卡卡面印有银联UPcash标识、持卡人姓名、有效身份证件号码及照片，卡内芯片中存有持卡人姓名、有效身份证件号码等实名信息。中铁银通卡根据铁路应用分为金卡和银卡。目前，持卡人凭中铁银通卡可以在铁路管理部门指定的区段内通过铁路车站自动检票机（闸机）直接刷卡乘车，还可使用中铁银通卡在铁路安装有POS机的售票窗口、支持银行卡支付的自助售票机、中国铁路客户服务中心网站（www.12306.cn，以下简称12306，以下简称12306网站）等渠道购票。

2、中铁银通卡是磁条、芯片双介质复合卡，其中磁条部分（联机账户）支持购票等联机消费功能，交易时须通过密码验证；芯片部分（电子现金）支持直接刷卡乘车等脱机消费功能，脱机交易时无须密码。

3、中铁银通卡仅限持卡人本人使用；中铁银通卡不取现、不计息；芯片有效期标注在卡片正面，过期后持卡人需到指定售卡网点办理换卡手续。

  申购

购卡时使用的有效身份证件范围为：中华人民共和国第二代居民身份证；港澳居民来往内地通行证；台湾居民来往大陆通行证；按规定可使用的有效护照。中国大陆居民购卡时仅支持中华人民共和国第二代居民身份证办理，不支持其他证件办理。购卡时交纳办卡押金30元/卡，退卡时退还押金。自卡片售出之日起第十二个月的月末系统自动从联机账户中扣缴年费10元/卡。

  充值

一、充值是指您将资金存入卡中，可以分别存入联机账户或电子现金，并可在规定限额内进行本卡内联机账户和电子现金之间的资金互转。充值金额为1元的整数倍。

二、联机账户资金限额为人民币5000元，电子现金限额为人民币1000元，同一卡片资金限额为人民币5000元。

三、充值时不开具发票，每次消费后由商户为您开具发票。充值渠道如下：

1、网点充值：中铁银通支付有限公司售卡网点、铁路网点窗口、中国银行网点使用现金进行充值。

2、中银自助通设备充值：您可以在中国银行网点内通过“中银自助通”设备，使用中国银行长城借记卡向中铁银通卡联机账户进行转账充值，请您按提示进行相应操作。

3、网上银行充值：您可以登录中国银行官方网站（http://www.boc.cn），使用中国银行长城借记卡或活期一本通账户向中铁银通卡联机账户进行充值。

4、手机银行充值：使用中国银行长城借记卡或活期账户向中铁银通卡联机账户进行充值。

5、签约自动转账充值：签约自动转账充值是指您将中铁银通卡联机账户与银行账户进行关联，按照您的要求，当中铁银通卡联机账户余额低于您设定的余额下限时，相关联的银行账户自动向中铁银通卡联机账户充入您设定的金额。　　您可以持本人的中铁银通卡、中国银行长城借记卡或活期一本通、有效身份证件到中国银行售卡网点办理自动转账签约业务，您在办理签约时所持身份证件须与银行账户开户时以及购买中铁银通卡时所使用的身份证件一致。

  使用

一、刷卡乘车

1、您持有效的中铁银通卡，可以在铁路管理部门指定的区段内刷卡乘车。“指定区段”以中铁银通支付有限公司网站及售卡网点的公告为准，且铁路管理部门有权根据其运营需要变更“指定区段”。

2、当您进站时，需持中铁银通卡通过铁路车站自动检票机（闸机）刷卡验证后进站。

3、您使用中铁银通卡进站、刷卡乘车时，可不再出示其他有效身份证件。

4、您上车后请到列车预留席就坐。

5、 您持中铁银通金卡刷卡乘车限乘坐一等座席位，并按一等座票价扣款；您持中铁银通银卡刷卡乘车限乘坐二等座席位，并按二等座票价扣款。

6、当您下车出站时，需要通过铁路车站自动检票机（闸机）刷卡验证，并扣减所乘区段票款后方可出站。

7、您刷卡乘车后，自乘车之日起31日内，可在铁路车站指定售票窗口补打车票，逾期不予办理。

8、每次乘车只能打印一次车票，不可重复打印。

9、您在刷卡乘车时卡片出现异常情况，请及时与铁路工作人员联系或到铁路指定窗口处理。

10、体刷卡乘车事项请您按铁路管理部门的相关规定执行。

二、购票

1、您持有效的中铁银通卡，可以在铁路安装有POS机的售票窗口、支持银行卡支付的自助售票机、12306网站等渠道购票。

2、购票交易以联机方式完成，实时从联机账户中完成扣款，交易时须验证卡片密码。

3、当卡片密码连续3次输入错误时，您的卡片会自动锁定，次日自动解锁。

4、您使用中铁银通卡在铁路售票窗口购票时，可不再出示其他有效身份证件。

  客户服务

一、换卡

1、请妥善保管您的中铁银通卡，并留意您所持有卡片正面标注的有效期。

2、如果您的卡片因受损或超过使用期限无法继续使用，需携带购卡时所使用的有效身份证件到铁路售卡窗口办理换卡业务。

3、如果因磁条损坏或卡片到期而换卡，原卡联机账户余额和电子现金余额均即时转入新卡的联机账户中；如果芯片损坏，补领新卡时，原卡联机账户余额转入新卡联机账户，新卡电子现金余额为零，原卡电子现金余额在您补领新卡之日起3日后，经确认自动转入新卡联机账户中。

4、如果您的卡片因损毁无法识别持卡人信息时，不能办理换卡，窗口工作人员将按卡片丢失处理。

二、退卡

1、当您需要退卡时，可在购买中铁银通卡3个月后，携带原卡及购卡时所使用的有效身份证件到铁路售卡窗口办理退卡业务。退款将以现金形式退还给您。

2、当您办理退卡业务时，您的卡片状态正常，可以直接办理退卡业务；您的卡片状态异常时，需要到铁路指定售卡窗口处理后，再办理退卡业务。

3、如果卡片芯片未损坏退卡，办理退卡业务后可即时领取退款；如果卡片芯片损坏，您应在铁路售卡窗口办理坏卡登记，待3日后领取退款。

4、当您的卡片丢失需要办理退卡时，您需到铁路售卡窗口办理正式挂失业务，再办理退卡业务。

5、办理退卡业务时，需要您交纳退卡费20元/卡。

6、如果您的卡片因损毁无法识别持卡人信息时，不能办理退卡，窗口工作人员将按卡片丢失处理。

三、挂失

中铁银通卡仅支持联机账户的挂失，不支持电子现金的挂失。当您发现中铁银通卡丢失或被盗，您可以致电客户服务热线（4008-368-368），进行卡片的临时挂失。临时挂失5日后将自动解除，卡片状态恢复正常。卡片丢失后或已经办理了临时挂失，请持购卡时所使用的有效身份证件到铁路售卡窗口办理正式挂失手续。

四、补卡

1、卡片正式挂失后办理补卡时，请您持购卡时所使用的有效身份证件到铁路售卡窗口办理补卡手续。

2、补卡需交纳补卡费10元/卡。

3、原卡内联机账户余额直接转入新卡中，原卡电子现金不办理挂失，新卡电子现金余额为零。

  客服热线

一、客户服务热线（4008-368-368）竭诚为您服务，及时为您解决在用卡过程中遇到的疑虑和问题，让您尽享一站式服务的快捷与方便。

二、您可拨打客户服务热线（4008-368-368），按照语音提示为您提供以下服务：

1、业务介绍      修改密码

2、信息查询      余额查询

3、交易查询      人工服务

4、临时挂失      客户建议或意见

  温馨提示

1、您初次办卡时，需交纳卡片押金，退卡时退还押金。

2、自卡片售出之日起第十二个月的月末系统自动从您的联机账户中扣缴年费，以后年费按每满十二个月进行扣缴。若您的联机账户余额不足，系统无法扣取年费时，您应在各代售网点补交年费或将年费存入联机账户中待系统自动扣取。
